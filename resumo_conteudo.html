Organização de Frases
Como montar uma frase com sentido
💡 O que é uma frase?
Uma frase é um conjunto de palavras que tem um sentido completo. Para fazer sentido, a frase precisa ter pelo menos um verbo e estar organizada na ordem certa.
📐 As partes da frase
QUEM?
Sujeito — quem faz a ação.Ex: A menina... / O violinista... / Os pais...
O QUÊ FAZ?
Verbo — a ação ou estado.Ex: ...tocou... / ...sorriu... / ...comprou...
O QUÊ?
Complemento — o que completa o sentido.Ex: ...o violino. / ...de alegria. / ...uma surpresa.
✅ Exemplos corretos
A menina tocou o violino.
Os pais aplaudiram a filha.
❌ Frase desorganizada
violino menina tocou o a
🧠 Macete
Sempre pergunta: Quem fez? → sujeito. O que fez? → verbo. O que / para quem? → complemento.

Sílaba Tônica e Acento Gráfico
A sílaba mais forte da palavra
💡 O que é sílaba tônica?
Em toda palavra com mais de uma sílaba, uma delas é pronunciada mais forte. Essa é a sílaba tônica! Ex: MÚ-si-ca, vi-o-LI-no, me-NI-na.
´
Acento Agudo
Usado em vogais abertas: a, e, i, o, u
músIca · café · avó
^
Acento Circunflexo
Usado em vogais fechadas: â, ê, ô
você · avô · tênue
📐 Classificação pelo acento
OXÍTONA
Última sílaba é a mais forte.Ex: ca-FÉ · ma-ra-cu-JÁ · tam-BÉM
PAROXÍTONA
Penúltima sílaba é a mais forte.Ex: me-NI-na · vi-o-LI-no · ca-SA · cha-PÉU
A maioria das palavras do português é paroxítona!
PROPAROXÍTONA
Antepenúltima sílaba é a mais forte.Ex: MÚ-si-ca, LÁ-gri-ma — sempre têm acento!
🧠 Macete
Bata palmas em cada sílaba e perceba qual soa mais forte. Essa é a tônica!

O livro mostra dois acentos gráficos:
• Acento agudo (´) = vogal aberta → mú-si-ca
• Acento circunflexo (^) = vogal fechada → vo-cê

Proparoxítonas sempre têm acento gráfico!

Substantivo Simples e Composto
Nomes formados por uma ou mais palavras
💡 O que é substantivo?
Substantivo é a palavra que dá nome às coisas, pessoas, lugares e sentimentos. Ex: violino, menina, música, alegria.
📐 Simples × Composto
SIMPLES
Formado por uma palavra só.Ex: violino · menina · flor · cachorro · violinista
COMPOSTO
Formado por duas ou mais palavras juntas.Ex: couve-flor · guarda-chuva · pé-de-moleque · vaivém · fim de semana
couve-flor
couve + flor → composto
violino
uma palavra → simples
pé-de-moleque
pé + de + moleque → composto
guarda-chuva
guarda + chuva → composto
menina
uma palavra → simples
girassol
gira + sol → composto
🧠 Macete
Conta as palavras que formam o nome.
Uma palavra = simples. Ex: violino, casa, flor
Duas ou mais palavras = composto. Ex: couve-flor, guarda-chuva, pé-de-moleque

Compostos podem ter hífen (couve-flor) ou ser uma palavra só (girassol).

Monossílabos Tônicos e Átonos
Palavras de uma sílaba só
💡 O que é monossílabo?
Monossílabo é qualquer palavra com uma sílaba só. Alguns são fortes (tônicos) e outros são fracos (átonos) na pronúncia.
📐 Tônico × Átono
TÔNICO
Tem sentido próprio e autonomia — pronunciado com mais força na frase.Ex: pó · pé · vó · Sol · chá · dá · pá · mim
ÁTONO
NÃO tem sentido próprio — precisa estar junto de outra palavra para ter sentido.Ex: de · a · que · com · do · o · em · se
Como identificar na prática
Tônicos (pintados de amarelo pela professora): pó · pé · vó · Sol · chá
Átonos (pintados de laranja): de · a · que · com · do

→ Tônicos têm sentido próprio: você entende a palavra sozinha!
→ Átonos precisam de outras palavras para fazer sentido.
🧠 Macete
A professora define assim:
Tônico = tem sentido próprio + autonomia + pronunciado com mais força.
Átono = NÃO tem sentido próprio — precisa de outra palavra para ter sentido.

💛 Dica da professora: tônicos = amarelo · átonos = laranja
Fale a palavra sozinha — se fizer sentido = tônico!

Substantivos e Verbos nas Instruções
Como funciona um texto de montagem
💡 O que é um texto de instrução?
É um texto que ensina a fazer algo passo a passo. Como montar um bilboquê ou uma tamanca. Usa verbos de comando e substantivos para nomear os materiais.
📐 Partes de um texto de instrução
TÍTULO
Nome do que vai ser feito.Ex: Bilboquê · Como fazer uma tamanca
MATERIAIS
Substantivos — lista de tudo que vai precisar.Ex: garrafa, fio, tampinha, fita adesiva
MODO DE FAZER
Verbos no imperativo — ordens e comandos.Ex: Corte · Cole · Faça · Amarre · Passe
Exemplo do livro — Bilboquê
Materiais (substantivos): garrafa pet, fita adesiva, tampinha, fio de náilon

Verbos de comando: Corte a garrafa. Faça um furo. Passe o fio. Cole a tampinha.
🧠 Macete
Na lista de materiais = substantivos (nomes das coisas). Nos passos = verbos de comando (o que fazer). Os verbos de instrução sempre começam o passo!
SS
Uso do SS
Quando usar SS nas palavras
💡 A regra principal
O SS aparece sempre entre duas vogais dentro de uma palavra. Uma letra S sozinha entre vogais teria som de Z, então precisamos de SS para manter o som de S.

⚠️ Atenção: A professora também trabalha S, X e Z — todos podem representar o mesmo som em diferentes palavras!
📐 A regra do SS
SS entre vogais
Vogal + SS + Vogal = som de SEx: a-sso-alho · ma-ssa · pa-ssar
Atenção!
S simples entre vogais = som de Z!Ex: ca-sa (caza) · ro-sa (roza) · a-sa (aza)
Por isso precisamos de SS para manter o som de S!
Família de palavras
Se uma palavra tem SS, a família mantém o SS.Ex: passa → passada → passagem → passador
S, X, Z
Outros exemplos da professora com som de S ou Z:aSSado · aZedo · eXercitar · pulXeira · penSSamento
EXaminar · aSfalto · aborrecido · aZarado · gaZolina
asso
vogal+SS+vogal
massa
vogal+SS+vogal
passar
vogal+SS+vogal
pressa
vogal+SS+vogal
esse
vogal+SS+vogal
osso
vogal+SS+vogal
🧠 Macete
O SS sempre fica entre duas vogais para ter som de S:
vogal + SS + vogal = som de S

Nunca use SS: no começo da palavra, depois de consoante.
Lembra: S entre vogais = som de Z · SS entre vogais = som de S

Adjetivos e Locuções Adjetivas
Palavras que caracterizam o substantivo
💡 O que é adjetivo?
Adjetivo é a palavra que caracteriza (descreve) um substantivo. Diz como ele é, como parece, de que tamanho, de que cor.
📐 Adjetivo × Locução Adjetiva
ADJETIVO
Uma palavra que caracteriza o substantivo.Ex: chinelo velho · chinelo grande · chinelo bonito
LOCUÇÃO ADJETIVA
Duas ou mais palavras (preposição + substantivo) que funcionam como adjetivo.Ex: chinelo de borracha · chinelo de pano · barco de madeira
Exemplos do livro — Tamanca
Adjetivos (uma palavra):
chinelo velho · sola plástica · luz solar

Locuções adjetivas (de + palavra):
chinelo de borracha · tampinha de garrafa · luz do sol · ambiente familiar → de família
🧠 Macete
A professora define: locução adjetiva = união de duas ou mais palavras que juntas exercem a função de adjetivo, servindo para qualificar ou caracterizar um substantivo.

Exemplo: cerebral (adjetivo) = do cérebro (locução adjetiva)
Exemplo: pluvial = da chuva · maternal = de mãe
